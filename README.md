# Student Result Management System

A web-based application built using Python Flask and MySQL to manage and display student academic results.

## Features
- Admin and Student login
- Add and update student marks
- View results with total and grade
- MySQL database integration
- Duplicate data prevention

## Tech Stack
- Python (Flask)
- MySQL
- HTML

## How to Run
1. Install Flask and MySQL connector
2. Create database using SQL script
3. Run `python app.py`
4. Open `http://127.0.0.1:5000/`
